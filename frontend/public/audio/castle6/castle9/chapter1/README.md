### Audio Format Requirements
- Format: MP3 (for smaller file size) or WAV (for quality)
- Sample Rate: 44.1kHz recommended
- Channels: Mono or Stereo
- Bit Depth: 16-bit (if using WAV)
- Bitrate: 192kbps or higher (if using MP3)

### Recording Guidelines
- Use an energetic, celebratory tone (this is the final assessment!)
- Keep background music triumphant but not overwhelming
- File naming follows zero-indexed convention (opening_0, opening_1, etc.)
- Make sure the audio celebrates the student's journey and progress
